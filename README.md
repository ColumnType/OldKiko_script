# OldKiko_script

This is the GitHub repo for the old Kikọ script. The font file can be used with any Yorùbá keyboard. However, we recommend **'Yoruba 8'** via the **Keyman** software and a **'Uncode Hex Input'** Keyboard for accessing nasal letters. 

##### *User will encounter encoding issues while attempting to use the font. Depending on the software and keyboard.
## Old Kikọ
**Old kikọ** is an alphabetic cursive script developed for writing standard Yorùbá. The script can be written left to right - horizontally, and right to left - vertically. The typeface includes a number of opentype features to be accessed with the layout editor of choice. **Further work on this script and typeface has been discontinued in favor of developing a new script.**
